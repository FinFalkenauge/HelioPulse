# Design System

## Theme Direction

Dark-first premium cockpit with strong contrast, layered depth, and chart-led storytelling.

## Color Tokens

- `bgDeep`: #090B10
- `bgRaised`: #111827
- `cardGlass`: rgba(22, 32, 52, 0.72)
- `solarAmber`: #FFB347
- `flowCyan`: #5CD9FF
- `stateGreen`: #39D98A
- `warnCoral`: #FF6B6B
- `textPrimary`: #F4F7FF
- `textSecondary`: #AAB6D3

## Typography

- Display numbers: Avenir Next Condensed, bold
- Labels and body: Avenir Next, regular and medium

## Components

- Glass KPI Card
- Mini Sparkline Strip
- Scenario Forecast Pill
- Data Confidence Badge

## Motion

- quick reveal for primary metrics
- subtle chart fade-in on range switch
- no decorative looping animation
